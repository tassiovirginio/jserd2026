# Artigos sobre Análise de Sentimentos em Mensagens de Commit

## Mais Citados (Fundamentação)

1. **Guzman, E., Azócar, D., Li, Y.** (2014). Sentiment analysis of commit comments in GitHub: an empirical study. *11th Working Conference on Mining Software Repositories (MSR)*. 365 citações.
   - https://dl.acm.org/doi/abs/10.1145/2597073.2597118

2. **Lin, B., Zampetti, F., Bavota, G., Di Penta, M. et al.** (2018). Sentiment analysis for software engineering: How far can we go? *40th International Conference on Software Engineering (ICSE)*. 289 citações.
   - https://dl.acm.org/doi/abs/10.1145/3180155.3180195
   - Questiona limitações da análise de sentimentos em commit messages

3. **Sinha, V., Lazar, A., Sharif, B.** (2016). Analyzing developer sentiment in commit logs. *13th International Conference on Mining Software Repositories (MSR)*. 147 citações.
   - https://dl.acm.org/doi/abs/10.1145/2901739.2903501

## 2020+

4. **Kaur, R., Chahal, K.K., Saini, M.** (2022). Analysis of factors influencing developers' sentiments in commit logs: insights from applying sentiment analysis. *e-Informatica Software Engineering Journal*. 11 citações.
   - https://www.e-informatyka.pl/attach/e-Informatica_-_Volume_16/eInformatica2022Art02.pdf
   - Focado em fatores que influenciam sentimentos

5. **Dey, K., Singh, J., Cao, H., Palma, F.** (2025). Pushing Feelings: Emotion and Sentiment in Software Commit Messages. *IEEE International Conference on Software Analysis, Evolution and Reengineering (SANER)*.
   - https://ieeexplore.ieee.org/abstract/document/11344467/
   - Guideline para análise de emoções/sentimentos em commit messages

6. **Huq, S.F., Sadiq, A.Z., Sakib, K.** (2020). Is developer sentiment related to software bugs: An exploratory study on GitHub commits. *IEEE International Conference on Software Analysis, Evolution and Reengineering (SANER)*. 44 citações.
   - https://ieeexplore.ieee.org/abstract/document/9054801/

7. **Singh, N., Singh, P.** (2017). How do code refactoring activities impact software developers' sentiments? An empirical investigation into GitHub commits. *24th Asia-Pacific Software Engineering Conference (APSEC)*. 33 citações.
   - https://ieeexplore.ieee.org/abstract/document/8305996/

## Complementary / Outros

8. **Souza, R., Silva, B.** (2017). Sentiment analysis of Travis CI builds. *IEEE/ACM 14th International Conference on Mining Software Repositories (MSR)*. 45 citações.
   - Commits com sentimento negativo levam a build quebrado?

9. **Patel, S., Zagade, G., Sharma, R.** (2024). Sentiment analysis of GitHub commit comments using machine learning techniques. *2nd DMIHER International Conference*.
   - https://ieeexplore.ieee.org/abstract/document/10842851/

10. **Kaur, R., Chahal, K.K.** (2020). Analysis of developers' sentiments in commit comments. *International Conference on Advanced Informatics for Smart Health*.
    - https://link.springer.com/chapter/10.1007/978-981-16-3660-8_1

## Notas

- **Para limitarções/metodologia**: Lin et al. (2018) e Kaur et al. (2022) discutem ameaças à validade
- **Para justificar uso**: Guzman et al. (2014) é o estudo seminal, muito citado
- **Para trabalho futuro**: Dey et al. (2025) é recente e traz guideline atualizado
